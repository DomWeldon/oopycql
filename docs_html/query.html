classmethod from_module(f, extension='.cql')[source]¶

Load a CQL query from the current module. By default, it is assumed that there is directory called CQL which contains the queries.

Parameters:
  • f – filename of the query
  • submodule – the directory name to load from, cql by default, if None then will use current module directory.
  • extension – the file extension (inc dot) of the query, if None then will assume no extension
